# E-Commerce-Sales-Analysis-
E-commerce sales analysis using SQL and Power BI

## Tools Used
- SQL
- Power BI
- Excel

## Dataset
The dataset contains order details, product category, customer city, and payment methods.

## Key Analysis
- Revenue by product category
- Top selling products
- Sales trend analysis
- Payment method distribution

## Dashboard
The Power BI dashboard provides interactive insights into sales performance.
